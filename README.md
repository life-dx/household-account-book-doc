# household-account-book-doc

家計簿アプリV2のためのドキュメント

## 概要

現行アプリはGoogle Formで作られているのでどこからでも入力できる。
しかし、どのみち家からしか入力しないため、あまり意味がない。
さらに、なかなかに入力しにくい。
ということで、Webアプリとして再構築する。

## 要件

- 支出がWebから簡単に入力, 閲覧, 編集できること
- スマホでも入力, 閲覧, 編集ができること
    スマホ: iPhone
    PC: ブラウザ解像度はFHD(1920 x 1080)
- 透明性確保のため、領収書などが添付できること。変更, ダウンロード, 閲覧できること
- 任意の期間で集計できること
- 毎月必ず入力する項目について、期日までに入力がない場合にリマインド通知を送信すること (エラー)
- 入力が必須と思われる項目について、期日までに入力がない場合はリマインド通知を送信すること (警告)
- Googleアカウントでログインできること
- 過去の入力をもとに自動入力できること
- 月次で割り勘のシートを出力し、通知を送信すること
- カテゴリごとの支出割合を集計できること
- カテゴリごとの支出割合の変化を月単位, 日単位で確認できること
- 月次の割り勘集計時にユーザーが確認していない場合はエラーとなり、確認依頼通知が送信されること
- 通知はラインで送信する
- 確定した支出項目は編集できないこと
- 将来的な拡張性を考慮し、割り勘はグループ単位での実行が可能とする
